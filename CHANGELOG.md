# CHANGELOG

## Version 120005.03

- Added profile import/export. Available at the "Profiles" tabs in Blizz config panel.

## Version 120005.02

- Fixed a memory leak caused by constant updates of trade post's and endeavours' data.

## Version 120005.01

- Added 12.0.5 support.
- Fixed an issue where 3rd party LSM textures wouldn't load correctly.
- Updated Simplified Chinese translation. Translated by 7peanuts@GitHub.

## Version 120001.01

- Added support for Travel Points. Use newly added "Show as Experience Bar" toggle above the progress bar in the
  Traveler's Log tab.
- Added support for Neighborhood Initiatives. Use newly added "Show as Experience Bar" toggle above the progress bar in
  the Endeavors tab.

## Version 120000.04

- Added options to fade in the monobar on entering combat and/or acquiring a target.

## Version 120000.03

- Added proper handling of Edit Mode layout copying, removal, and deletion.
- Reduced the min width to 384px.
- Fixed an issue where the monobar wouldn't hide itself while in a vehicle or during a pet battle.

## Version 120000.02

- Fixed an issue where fading wasn't applied on load.

## Version 120000.01

- Release released.
