# LS: Monobrow

A replacement for the default xp, reputation, etc. bars. And just like your eyebrows it's all in one!

![Imgur](https://imgur.com/5Cnfw7a.gif)

## Download

- [Curse](https://www.curseforge.com/wow/addons/ls-monobrow)
- [Wago](https://addons.wago.io/addons/ls-monobrow)

## Options

Most settings are available via the edit mode to support layouts, the rest can be accessed via the addon compartment.

## Feedback and Feature Requests

If you found a bug or want to share an idea on how to improve my addon, either use the issue tracker on [GitHub](https://github.com/ls-/ls_Monobrow/issues) or join our [Discord](https://discord.gg/7QcJgQkDYD) server.

## Localisation

Feel free to add and/or review translations on [CurseForge](https://www.curseforge.com/wow/addons/ls-monobrow/localization), alternatively, you may create a PR on [project's GitHub page](https://github.com/ls-/ls_Monobrow/pulls).
